# Scientific-Calculator_GUI
**GUI based Scientific calculator**


Built using JAVA Swing and AWT featuring: 

Standard arithmetic operations: + - * / 

Trigonometric fucntions: (sin cos tan cot sec csc, and their inverses) + DEG <-> RAD 

Advanced fucntions: sqr sqrt fac abs exp log(natural log and ln) pow 

Additionals: Binary, Octal, Hexdecimal bases conversions 

2nd toggle switch: log/ln/10^x <-> binary/octal/hex Has error handling

Improvement to be done for the functionalities implementation of Memory operations (MC, MR, M+, M-, MS)

Note: Since this code uses ScriptEngine, it is very well compatible with the recent JAVA versions, and suits bets with JDK 8 - 10.
